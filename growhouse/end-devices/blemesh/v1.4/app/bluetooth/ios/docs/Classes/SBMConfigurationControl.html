SBMConfigurationControl

@interface SBMConfigurationControl : NSObject

SBMConfigurationControl methods are for configuring a node over the mesh network by sending configuration model messages. Note that the API is asynchronous; unless there is an immediate local failure. Important: It is not needed to keep object by application. Framework will do it.
